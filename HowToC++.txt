


How to compile a C++ program:

g++ Source/Main_PerfCode2020.cpp -o fw
compiles Main_PerfCode2020.cpp to fw


./fw
executes fw

How to run python wake plotting. Run from main folder
g++ Source/Main_PlotWakePY.cpp -o PlotWakePY
./PlotWakePY on linux or PlotWakePY if on windows


How to run Qiver

g++ Source/Main_PlotQuiver20PY.cpp -o output/QuiverFW20

will write executable into output directory.
QiverFW20 has to be run out in the output directory using ./QuiverFW20 or output/QuiverFW20



How to run Vogen (generates input file for fw)
in directory freeWakeImpDev_D2C

compile in VoGenDevelop
gcc -c *.c
gcc -o vogen -lm *.o
or
gcc *.c -lm -o vogen
with "vogen" name of executable, needs to be copied into freeWakeImpDev_D2C

./vogen predesignD2C.inp
